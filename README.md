# Car Service Management System

A simple Java console application for managing a car service center.

## Features

- Register incoming cars
- Track service status (In Service / Serviced)
- View car details
- Manage service capacity (max 56 cars per day)
- Generate invoice with multiple items
- Display daily greeting message based on current day

## Technologies Used

- Java
- Console-based interaction
- LocalDateTime API

## How to Run

1. Clone the repository
2. Open in your preferred IDE (IntelliJ, Eclipse, etc.)
3. Run the `ession1` class
